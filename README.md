## Building Locally

### On Termux
```console
bash <(curl -sSf https://raw.githubusercontent.com/LyelKwon1/Youtube-Black/main/build-termux.sh)
```

### On Linux
```console
$ git clone https://github.com/LyelKwon1/Youtube-Black --depth 1
$ cd Youtube-Black
$ ./build.sh
```
